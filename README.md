# Computer Vision Course
Academic course of Computer Vision with OpenCV <br>
Lecturer: Ruslan Jantayev

## Prerequisites
* Python
* OpenCV
* Numpy
* Matplotlib (Optional)


## Installation
Simply clone the repository or download zip: 
```sh
$ git clone git@github.com:ErsainKRX/opencv-python-course.git
```
